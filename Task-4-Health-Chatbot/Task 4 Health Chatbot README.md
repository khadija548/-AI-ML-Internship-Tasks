# Health Assistant Chatbot 

## Task Objective
Build an AI-powered chatbot that answers general healthcare questions using natural language processing.

## Technologies Used
- Python
- Streamlit
- Hugging Face Transformers
- FLAN-T5 model

## Model Used
google/flan-t5-base

The model generates structured responses to health-related questions.

## Features
- Chat interface
- Health question answering
- Structured responses
- Chat history
- Emergency safety filter
- Medical disclaimer

## Example Questions
- What is ICU?
- What causes fever?
- Symptoms of dengue
- What is asthma?

## How to Run

Install dependencies:
pip install -r requirements.txt
Run the chatbot:
streamlit run app.py
